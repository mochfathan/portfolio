🎓
Domain-First Design
Each tool was designed from the domain outward — the financial ratios, Indonesian COA codes, PPh 23 rules, and S&D odds enhancers came first. The AI was the delivery mechanism, not the knowledge source.
⚡
Production-Grade Output
No demo data, no hardcoded responses. Every tool calls the Claude API live, parses structured JSON, and renders results a real practitioner could act on. PAJAKPINTAR generates legally accurate journal entries. TRADECRAFT enforces real methodology rules.

Every tool is a self-contained HTML file. No framework, no build step, no server. Hosted free on GitHub Pages. The architecture constraint forced discipline: no bloat, no dependencies, maximum portability.
